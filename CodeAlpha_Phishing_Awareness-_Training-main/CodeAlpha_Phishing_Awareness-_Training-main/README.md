# CodeAlpha_Phishing_Awareness-_Training
Phishing is one of the most common cyberattacks used to steal sensitive information such as login credentials, financial data, and personal information. This presentation explains:

*What phishing is

*Types of phishing attacks

*Common phishing email red flags

*Social engineering tactics used by attackers

*Real-world phishing examples

*Best practices to avoid phishing attacks

*Steps to take if compromised

*Interactive quiz for learning reinforcement
